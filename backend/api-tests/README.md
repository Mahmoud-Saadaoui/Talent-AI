# API Tests

Ce dossier contient des fichiers `.http` pour tester les API manuellement via l'extension VS Code **REST Client**.

## Prérequis

- Installer l'extension [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client) pour VS Code.

## Utilisation

1. Ouvrez un fichier `.http`.
2. Cliquez sur "Send Request" au-dessus de chaque requête.
3. La réponse apparaîtra dans un nouvel onglet.
